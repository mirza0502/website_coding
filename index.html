HTML

HTML (Hypertext Markup Language), merupakan salah satu bahasa pemograman yang digunakan untuk membuat halaman website pada web browser.

CSS

CSS (Cascading Style Sheet) biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.

JavaScript

JavaScript digunakan untuk membuat website interaktif dan dinamis. Adapun dinamis yang dimaksud disini berarti konten di dalamnya dapat bergerak dan tampil secara otomatis tanpa harus dimuat ulang manual oleh pengguna.
